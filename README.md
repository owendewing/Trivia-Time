# Trivia-Time
Trivia-Time is a project I worked on when I took my Mobile Application Development course with a partner. This app was developed with Swift and allows users to login/create an account, play a quick trivia game, and uses Firebase/Firestore to send their score to a leaderboard where they can compare their score to other players' scores. We used a Trivia API to generate 10 questions from random categories and we calculate the users' score by combining the amount of time left with the correct/incorrect answer. Some features that could be developed in the future could be 50-question or 100-question trivia games, trivia games based on category (sports, history, etc), and more collaborative features (following your friends, battle modes, etc). I included a software demo video to display a sample user of Trivia Time.
