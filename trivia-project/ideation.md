Owen Dewing, Thomas Powell, Axel Pestoni 

 

App Ideation ideas: 

Choose an Audience: music listeners, travelers, athletes, movie watchers, college students, computer science students, family & friends 

 

Needs for audience:  

Family & friends:  educate children, a way to have fun together, competition between families, interaction with friends, community building, learning more about each family, bond over common interests, movie night, meeting new people, music night, book club 

 

Map needs to projects:  

Educate children: trivia game 

Meeting new people: mobile card game or mobile charades game 

Movie night: movie picker w/ filter by genre 

 

Produce Goals/Ideas for each project: 

Trivia: different categories of questions, multiplayer, timer, point system, trivia API implementation, difficulty adjustment, pick amount of players 

Mobile card game: pick amount of players, team-game, separated by rounds, point system 

Movie picker: a random spinner, a way to filter by genre, rating, length, etc., ability to create an account with the movies that you have picked from the app 

 

Winner: Trivia game
