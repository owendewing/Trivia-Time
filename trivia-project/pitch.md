Inspiration Statement:  

Our app is a trivia app that helps family and friends compete with each other and learn new information. 

 

Viable solutions we were considering:  

We are considering using a trivia API, keeping scores in a Firebase database, signing in through email or game center, wanting the game to be multiplayer, adding a scoreboard/leaderboard system, etc. 

 

Feedback/New Ideas:  

    Scope is too large; make game single player 

    Create a set number of questions but make them random. Player receives more points the faster they get the question right. Try to answer as many questions as possible correctly in the lowest amount of time.  

    Could create a game mode that only gives questions from one category if people have more knowledge in a certain category. 

    Use firebase to track the previous highest score by the player and display it at the top when the player is answering questions. This will incentivize the player to try and beat their previous high score. 

    Figuring out how to use a timer (we found a timer class through Swift) 

 

Final app story: Our app is a trivia app that allows family and friends to compete with each other to answer the greatest number of questions in the least amount of time and learn new information.
